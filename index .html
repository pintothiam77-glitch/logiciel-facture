Analyse du marché local
⚠️ Le contenu financier est généré par IA (1000% IA) à titre indicatif et doit être vérifié par un professionnel avant toute décision d'investissement ou de financement. Chaque génération de fiche nécessite un paiement de 15 000 FCFA via Wave : la fenêtre de paiement affichée est une maquette d'interface, aucune transaction réelle n'est traitée — l'intégration effective nécessite une clé API PayDunya / UnitechPay / PayTech côté serveur (voir dossier /backend pour brancher un vrai webhook de paiement). La gestion des comptes, rôles et clés API peut fonctionner en mode prototype (stockage d'artefact) ou en mode serveur réel (Node.js + Express + SQLite, bcrypt, sessions par cookie) selon que window.SBAI_BACKEND_URL est défini.
